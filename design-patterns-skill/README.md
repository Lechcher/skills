# Design Patterns Skill

Agent skill that provides automated advice, documentation, and refactoring strategies based on the 22 GoF design patterns compiled from the Refactoring Guru catalog.

## Installation

Run the provided installation script:

```bash
./install.sh
```

## Usage

In any supported agent framework (Cursor, Windsurf, Claude Code, Gemini CLI, Antigravity, etc.), invoke the skill by typing:

```
/design-patterns-skill Evaluate this class and suggest a creational pattern if applicable.
```
