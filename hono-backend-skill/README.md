# Hono Backend Skill

This is a Level 5 Agent Skill for developing HonoJS backends. 
It provides the agent with documentation and structure to bootstrap, develop, and architect Hono applications confidently, including routing, middleware, and context handling.

## Installation

You can install this skill on any supported AI coding platform (Claude Code, Cursor, Windsurf, Copilot, etc.):

```bash
git clone <your-repo-url> ~/.agents/skills/hono-backend-skill
```

Or download the package and run:

```bash
cd hono-backend-skill
./install.sh
```

## Usage

Once installed, invoke the skill in your AI assistant:

```
/hono-backend-skill Create a new Hono project for Cloudflare Workers
/hono-backend-skill Add a JWT authentication middleware to my API
/hono-backend-skill Show me how to use Hono Context to read headers
```
