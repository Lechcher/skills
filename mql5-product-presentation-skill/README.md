# mql5-product-presentation-skill

A Level 5 skill providing guidelines and tips for an effective product presentation on the MQL5 Market. Helps design logos, screenshots, and descriptions for MetaTrader Expert Advisors and indicators.

## Installation

### Mac / Linux
```bash
git clone <your-repo> ./mql5-product-presentation-skill
cd mql5-product-presentation-skill
chmod +x install.sh
./install.sh
```
